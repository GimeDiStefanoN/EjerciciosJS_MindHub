Ejercicios de Condicionales

Resumen

  1. 1. Palabra reservada “if” para crear una sentencia de decisión.
  2. 2. Abrir paréntesis (expresión a evaluar) donde ira la sentencia a evaluar la cual dará siempre tendrá como valor verdadero (true) o falso (false).
  3. 3. Abrir llaves {… bloque de código…} para escribir la o las acciones a realizar en caso de que la expresión a evaluar halla sido verdadero (true).
  4. 4. Luego de la llave de cierre de la expresión “if” se puede utilizar “else” por si se quiere realizar alguna acción o acciones cuando la expresión evaluada dentro del “if” halla sido falsa (false).
  5. 5. En caso de usar “else” se debe abrir nuevamente llaves {…bloque de código…} para realizar la o las acciones en el mismo.
  6. 6. También se puede escribir “else if {…bloque de código…}” para realizar otra pregunta en caso de haber sido falsa (false) el valor del “if” inicial y se puede seguir agregando tantos “else {}” o “else if{}” como se requieran al finalizar una sentencia “if” o “else if”.
  7. 7. Palabra reservada “switch” para crear una sentencia de decisión múltiple.
  8. 8. Abrir paréntesis (variable de control) para colocar la variable que se va a evaluar el valor que contiene.
  9. 9. Abrir llaves para escribir el bloque de código del “switch(variable de control){…bloque de código…}”
  10. 10. Dentro de las llaves se definen los casos literales a evaluar: ”case ‘valor’:{}”.
  11. 11. Dentro de cada caso se debe usar la palabra reservada “break” para finalizar ese caso de uso.
  12. 12. Al final de los casos declarar un caso “default:{}” para cualquier caso no contemplado.
  13. - Dato extra, pueden utilizar la función “confirm(‘texto de pregunta al usuario’)” para poder preguntar algo por pantalla y según sea la respuesta escogida por el usuario la función devuelve verdadero (true) o falso (false) según corresponda.
